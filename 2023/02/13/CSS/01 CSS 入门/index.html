2 如何构建 CSS

2.1 在 HTML 中应用 CSS

2.1.1 外部样式表

  • 外部样式表是指将 CSS 编写在扩展名为.css 的 单独文件 中，并从 HTML <link> 元素引用它

2.1.2 内部样式表

  • 内部样式表是指 不使用外部 CSS 文件，而是将 CSS 放在 HTML 文件<head> 标签里的<style> 标签之中
  • 低效

2.1.3 内联样式

  • 内联样式表存在于 HTML 元素的 style 属性之中，特点是只影响单个元素
  • 难以维护

2.2 选择器

  • 选择器示例

专一性

  • 当两条 CSS 规则发生碰撞时，哪条规则起效？

2.3 属性和值

  • 属性：标识符，指示希望更改的央视特征
  • 值：指示如何更改
  • ⚠ 在 CSS 中，属性和值都是区分大小写的
  • ⚠ 如果属性未知或某个值对给定属性无效，则声明被视为无效，并被浏览器的 CSS 引擎完全忽略

calc() 函数

  • 作用：允许在 CSS 中进行简单的计算

transform 属性

  • transform: rotate(0.8turn) 旋转

2.4 @规则

  • @import 'styles2.css'
  • @media，允许使用媒体查询应用 CSS（当条件成立时应用该规则）

2.5 速记属性

  • 速记属性：允许在一行中设置多个属性值
  • font
  • background
  • 四值设置：上、右、下、左（顺时针）
    • padding: 10px 15px 15px 5px;
    • border
    • margin
  • 二值设置：上下、左右

2.6 注释

  • /* ...... */

2.7 空白

  • 多个语句之间的空白
  • 单个语句内的空白
    • 通过至少一个空格将不同的值分隔开来
    • 将属性名称和属性值作为单个未中断的字符串放在一起

3 CSS 运行原理

3.1 CSS 工作基本流程

1. 浏览器载入 HTML 文件 2. 将 HTML 文件转化成一个 DOM（Document Object Model） - DOM 是文件在计算机内存中的表现形式 3. 【拉取资源】浏览器会拉取该 HTML 相关的大部分资源，比如嵌入到页面的图片、视频和 CSS 样式。JavaScript 则会稍后进行处理 4. 【解析 CSS】 1. 根据选择器的不同类型将他们分到不同的“桶”中 2. 将不同的规则应用在对应的 DOM 节点中 3. 添加结点依赖的样式（渲染树） 5. 上述的规则应用于渲染树之后，渲染树会依照应该出现的结构进行布局。 6. 网页展示在屏幕上（这一步被称为 着色）

3.2 DOM

  • 一个 DOM 有一个 树形结构
  • 标记语言中的每一个元素、属性以及每一段文字都对应着结构树中的一个节点（Node/DOM 或 DOM node）
  • 节点由节点本身和其他 DOM 节点的关系定义，有些节点有父节点，有些节点有兄弟节点（同级节点）
