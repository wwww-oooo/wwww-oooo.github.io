03 HTML 表格

1 HTML 表格基础

1.1 最简单的表格

  1. 每一个表格的内容都包含在 <table></table> 之中
  2. 表格中最小的内容容器时单元格 <td></td>，(td: table data)，每个单元格会和同一行的单元格对齐
  3. 控制表格各行 <tr></tr> (tr: table row)

1.2 表格小标题（行、列标题）

  • <th> 表示表格标题 (th: table header)

1.3 合并单元格

  • colspan 属性和 rowspan 属性，该属性大小决定其宽度/高度是几个单元格

1.4 为表格中的列提供共同样式

  • 操作方法
  • 解读：用 <colgroup> 定义每一列的样式，每一个 <col> 代表其中一列，如果不设置则单独写一个空的 <col> 元素
  • 如果多列用同一个样式：
    • 只使用一个 <col> 元素，但包含 span 属性，设置应用这个样式的列数

2 高级表格特性和无障碍

2.1 表格标题 <caption>

  • 放置位置：<table> 标签下面

2.1 <thead>、<tfoot> 和 <tbody>

  • <thead> 放在 <colgroup> 后面，然后 <tbody>，然后 <tfoot>
  • <tbody> 不指定的话就是隐式指定的

2.2 无障碍

使用行、列标题

scope 属性

  • scope="row", scope="col"
  • scope="rowgroup", scope="colgroup"

id 和标题属性

  1. 为每个 <th> 元素添加一个唯一的 id
  2. 为每个 <td> 元素添加一个 headers 属性，每个 headers 属性包含它从属于的所有标题的 id，之间用空格分隔开

【理解】标注每个单元格都属于哪些行、列标题
